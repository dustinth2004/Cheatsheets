# Penetration Testing and Security Cheatsheets

This repository is a collection of cheatsheets for penetration testing, security, and various other technical topics. These cheatsheets have been collated over the years and are intended to be a quick reference for security professionals, system administrators, and developers.

## Cheatsheets

The cheatsheets are organized into the following categories:

### Operating Systems

*   [AIX Build](Cheatsheet_AIXBuild.txt) - Cheatsheet for AIX system administration.
*   [Linux Privilege Escalation](Cheatsheet_LinuxPrivilegeEsc.txt) - Techniques for escalating privileges on Linux systems.
*   [Solaris](Cheatsheet_Solaris.txt) - Cheatsheet for Solaris system administration.
*   [Windows Command Line](Cheatsheet_WindowsCommandLine.pdf) - Cheatsheet for the Windows command line.

### Networking

*   [Networking](Cheatsheet_Networking.txt) - General networking commands and concepts.
*   [Vlans](Cheatsheet_Vlans.txt) - Cheatsheet for VLANs.
*   [Cisco IOS](Cheatsheet_CiscoIOS.txt) - Cheatsheet for Cisco IOS commands.
*   [VOIP](Cheatsheet_VOIP.txt) - Cheatsheet for Voice over IP.
*   [SMB Capture](Cheatsheet_SMBCapture.txt) - Cheatsheet for capturing SMB traffic.
*   [SMB Enumeration](Cheatsheet_SMBEnumeration.txt) - Cheatsheet for enumerating SMB.
*   [SMTP Open Relay](Cheatsheet_SMTPOpenRelay.txt) - Cheatsheet for testing SMTP open relays.
*   [SSL Strip](Cheatsheet_SSLStrip.txt) - Cheatsheet for using SSLStrip.
